Masculinity Discourse Debate System

AI-Generated Debates Between Public Intellectuals on Masculinity

Bachelor's Thesis Project — University of Tartu, Institute of Computer Science

About This Project

This system stages structured debates between AI agents representing 14 public intellectuals. Each agent draws from the speaker's actual YouTube transcripts using retrieval-augmented generation (RAG), filtered for faithfulness to their real positions. Debates follow a 3-round structure: foundational beliefs, challenges and provocations, then rhetorical evidence.

Select a research question below to explore the debates, including full transcripts, moderator analysis, and synthesis of each exchange.
